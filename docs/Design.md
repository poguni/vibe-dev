# Dodge It - UI/UX 디자인 가이드 (Design Guide)

이 문서는 초등학교 3~6학년 학생들이 크롬북을 활용하여 단원 복습을 진행하는 '똥 피하기 게임(Dodge It)'의 디자인 톤 앤 매너와 시각적 가이드라인을 정의합니다.

---

## 1. 디자인 톤 앤 매너 (Tone & Manner)
- **컨셉 (Concept)**: 발랄하고 장난기 넘치는(Playful), 하지만 학습에는 방해되지 않는 깔끔함(Clean).
- **시각적 스타일 (Style)**: 둥근 외곽선을 가진 귀여운 '두들(Doodle) 스타일'과 캐주얼한 '픽셀 아트(Pixel Art)'의 조화. '똥 피하기'라는 코믹한 소재를 아이들이 시각적 거부감 없이 즐겁게 받아들일 수 있도록 최대한 둥글둥글하고 친근한 그래픽을 지향합니다.
- **핵심 감성 (Mood)**: 유쾌함, 성취감, 편안함.

## 2. 타이포그래피 (Typography)
아이들의 흥미 유발(게임성)과 문제 가독성(학습성)의 밸런스를 맞추기 위해 두 가지 폰트를 조합하여 사용합니다.

### 2.1 제목 및 장식용 폰트: **쿠키런체 (CookieRun)**
- **특징**: 동글동글하고 통통 튀는 귀여운 느낌으로, 게임 특유의 발랄한 분위기를 확실하게 연출합니다.
- **적용처**: 대제목(`<h1>`), 중제목(`<h2>`), 게임 로고 타이틀, 팝업창 제목, '게임 시작'과 같은 큼직한 버튼 내 텍스트.

### 2.2 본문 및 문제용 폰트: **프리텐다드 (Pretendard)**
- **특징**: 자간과 행간이 아름답고 가독성이 극대화된 모던 고딕 폰트입니다. 크롬북의 다소 작은 화면이나 낮은 해상도에서도 피로감 없이 글을 또렷하게 읽을 수 있습니다.
- **적용처**: 학습 문제 텍스트(가장 중요), 정답 입력창, 통계 수치 표기, 일반 설명문, 교사 대시보드 텍스트.

## 3. 컬러 팔레트 (Color Palette)
눈이 편안하면서도 아이들의 긍정적인 감정을 이끌어내는 색상을 사용합니다.

- 💛 **Primary (주조색)**: `활기찬 노란색 (#FFD54F)`
  - 긍정적이고 즐거운 에너지를 줍니다. 주요 버튼이나 포인트 요소에 사용합니다.
- 💚 **Secondary (보조색)**: `성취의 연두색 (#81C784)`
  - 정답을 맞혔을 때나 성공적인 이벤트를 나타내는 컬러로 활용하여 성취감을 부여합니다.
- 💩 **Accent (강조색)**: `부드러운 초코 브라운 (#8D6E63)`
  - 핵심 오브젝트(똥) 및 바닥(땅) 지형을 표현할 때 사용합니다. 너무 탁하거나 리얼하지 않은 부드러운 톤으로 엽기적인 느낌을 순화합니다.
- ☁️ **Background (배경색)**: `포근한 크림/파스텔 하늘색 (#FDFBF7, #E1F5FE)`
  - 게임의 배경 톤입니다. 눈의 피로를 덜고, 위에 올라가는 문제 텍스트의 가독성을 방해하지 않도록 채도를 낮춥니다.
- 🖤 **Text (텍스트)**: `다크 차콜 (#37474F)`
  - 완전한 흑색(#000000) 대비 눈의 피로도를 낮춰주는 짙은 회색을 문제 텍스트에 적용합니다.

## 4. UI 요소 및 레이아웃 (UI Elements & Layout)
- **크롬북 최적화 (16:9 비율)**: 크롬북의 표준 화면비에 딱 맞도록 레이아웃을 고정(Letterboxing 기법)하여, 어떤 모니터에서도 게임이 잘리거나 UI가 무너지지 않게 설계합니다.
- **집중형 문제 패널**: 게임 화면 캔버스 상단이나 우측에 '반투명한 하얀색 둥근 패널(Glassmorphism)'을 배치하고, 그 위에 문제를 출력합니다. 뒤쪽의 화려한 게임 그래픽과 철저히 분리하여 아이들이 문제를 읽는 데 100% 집중할 수 있게 만듭니다.
- **버튼 디자인**: 모서리가 둥글고 입체감 있는 '젤리 형태'의 버튼. 마우스로 올리거나 클릭할 때 쏙 들어가는 통통 튀는 애니메이션(Bouncy effect)을 적용합니다.

## 5. 애니메이션 및 시각적 피드백 (Visual Feedback)
즉각적이고 과장된 피드백을 통해 아이들의 몰입을 돕습니다.

- **정답 (Correct)**: 연두색 반짝임 파티클 터짐 + 경쾌한 팝(Pop) 사운드 + 똥 객체 사라짐.
- **오답 / 피격 (Wrong/Hit)**: 화면 테두리가 살짝 붉어지며 가볍게 흔들리는(Camera Shake) 효과 + 띠용~ 하는 코믹 사운드.
- **이벤트 - 변기 물 내리기**: 화면 위에서 아래로 시원한 파란색 물결(Wave) 그래픽이 휩쓸고 지나가며 화면을 청소하는 통쾌한 연출.
- **이벤트 - 변비**: 화면 내 똥 오브젝트들이 얼어붙은 채 부들부들 떨리는(Jitter) 코믹하고 우스꽝스러운 애니메이션.

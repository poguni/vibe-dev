[인공지능 챗봇(VibeChat) 프롬프트]


[프롬프트 #1] 인공지능 챗봇 프로토타입 만들기
  AI랑 채팅을 나눌 수 있는 일종의 챗봇 웹 앱을 만들어 주세요. 사용자는 로그인 없이 사용할 수 있어야 합니다.

Gemini API Key가 필요하면 얘기하고, Gemini 모델 중 gemini-flash-latest를 사용해 주세요.

----------

[수정]

1. 설정 창에서 새로운 'AI 역할 및 페르소나 설정'을 추가할 수 있도록 기능을 만들어 주세요. (예. AI 역할 - 통계학자, 시스템 프롬프트 - 당신은 유명한 통계학자입니다...)

2. API Key를 제외한 대화 내용과 설정 내용을 저장할 수 있으면 좋겠습니다. JSON 등의 파일로 저장해서 다른 컴퓨터에서 불러오면 대화나 설정을 이어서 사용할 수 있도록 해 주세요. JSON 말고 더 좋은 저장 방법이 있으면 제안해 주셔도 좋습니다.

3. 설정에서 API Key를 입력하고 저장하면 자동으로 화면이 리프레시 되어 API 연결 상태로 바꾸어 주세요.

[페르소나 예시]

  당신은 통계학을 전공하고 계량 통계학과 데이터 분석 분야에서 뛰어난 성과를 보인 대학 교수입니다. 대학에서는 주로 구조방정식모형, 다층모형, 종단분석 등을 강의하고 초보자부터 중급자까지 다양한 계층의 사람들이 통계분석을 쉽게 익힐 수 있도록 여러 권의 책도 저술했습니다. 최근에는 Python, Jupyter Notebook, Pandas, Numpy, Seaborn, Matplotlib 등의 도구를 이용해서 데이터 분석을 쉽게 배우고 응용할 수 있는 온라인 강의도 진행하고 있습니다.

[대화 예시]

  요즘 바이브 코딩이 인기인데 초급자도 쉽게 배울 수 있는 바이브 코딩 도구를 몇 가지 추천해 주세요. 우선은 무료로 사용해 볼 수 있는 서비스로 부탁합니다.

[프롬프트 #2] 음성 인식 기능 만들기
  음성 인식 기능을 추가해 주세요. 사용자의 음성 입력이 끝나면 자동으로 음성 인식 모드가 종료되도록 기능을 만들어 주세요.

[수정] 음성 인식 기능에서 말하다가 중간에 잠깐만 쉬어도 음성 인식이 중단됩니다. 음성 인식이 너무 빨리 끊기지 않도록 조정해 주세요.

----------

[프롬프트 #3] 답변을 읽어주는 기능 만들기
  인공지능이 해 준 답변도 음성으로 듣고 싶어요. Text-to-speech 기능을 구현해 주세요.

[수정] 음성 듣기 버튼이 너무 작습니다. 눈에 잘 띄도록 약간 크게 해 주고, 현재 읽고 있는 부분에 맞게 음성 on/off 기능이 따라 다니면 좋겠습니다.

----------

[프롬프트 #4] PDF 파일을 분석해서 답하는 기능 만들기
  PDF 파일을 업로드하면 그 파일 안에 있는 정보를 가져와 답변에 참고하는 기능을 추가해 주세요.

----------

[프롬프트 #5] 대화나 PDF 파일을 읽고 내용을 요약해서 리포트를 만드는 기능
  대화 내용을 요약해서 리포트를 만드는 기능을 구현해 주세요. 또, PDF 파일 내용을 분석해서 중요한 정보를 요약해 리포트를 만드는 기능도 필요해요. 요약 결과는 별도의 모달 창 위에 띄워 주고, 사용자가 원하면 인쇄할 수 있도록 버튼도 만들어 주세요.

----------

[프롬프트 #6] 인터넷 검색을 통해 자료를 찾아주는 기능 만들기(RAG)
  채팅 입력창 옆에 돋보기 버튼(검색하기 버튼)을 하나 만들어 주세요. 버튼은 처음에는 회색이고, 클릭해서 활성화시키면 컬러로 변하게 해 주세요. 버튼이 활성화된 상태로 질문하면 Gemini API가 구글 검색 그라운딩을 사용해 답변을 만들도록 해 주세요.

[수정] 채팅창의 버튼 순서를 
  1. 검색하기(돋보기) 버튼
  2. PDF 파일 추가(클립) 버튼
  3. 음성 인식(마이크) 버튼
  4. 메시지 입력 창
  5. 실행(비행기) 버튼
  순서로 바꾸어 주세요.
